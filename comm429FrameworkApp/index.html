Home Page

Welcome to my static site!
This site was built using 11ty, Tailwind CSS, and daisyUI.
It has three main pages, this home page, a page to view yugioh cards, and a blog directory.



The home page is just this, from it the other pages can be reached, not much else to say here.


The next page can be used to view yugioh cards.
The page has a search bar, that when typed into, will generate links to cards whose names match the query.
Each card has its own page that can be reached by clicking the card name.
On these pages information about the card is displayed, such as the card's name, status on the banlist, effect text, and an image.
To avoid making too large of a site, I limited the pool of cards to only those on the current banlist.
For some example cards to search for, try "pot of greed", or "monster reborn".
To vist that page, click the button below or use the links in the header.




The final page is the blog directory, it contains links to some blog posts.
Here I've added all of the blog posts I've made in the course, as well as some filler posts that I made while creating/testing the site.
To go to that page, click the button below or use the links in the header.
